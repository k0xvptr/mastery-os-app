# Mastery os app - Hackathon Web App

A lightweight, functional web application built in **36 hours** during a fast-paced hackathon challenge. This project focuses on making memorization easier

This is a project that's being built to complete school hackason with topic "education"
Our first project!!!

## 🛠️ Tech Stack & Architecture
*   **Backend:** Python & Flask, GO (RESTful Routing, Session Management)
*   **Frontend:** TypeScript, JavaScript (Clean, responsive user interface)
*   **Version Control:** Git & GitHub (Agile collaboration)

## ✨ Core Features
*   **Feature 1:** Generate questions with dynamic diffculties using AI
*   **Feature 2:** Save the questions and provide user feedback
